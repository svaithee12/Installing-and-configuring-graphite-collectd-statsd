This will guide you to configure the statsd PLUGIN of collectd to send the client machine data to statsd at server.



1. open the collectd config file 

	sudo nano /etc/collectd/collectd.conf

	if not that, 
		sudo nano /etc/collectd.conf

2. un-comment the LoadPlugin statsd

3. In the definition of statsd LoadPLugin 

	<Plugin statsd>
 		 Host "ip_address_of_server_machine"
  		 Port "8125"			// make sure the statsd service is running at port number 8125 on the server
		 DeleteCounters false
		 DeleteTimers   false
		 DeleteGauges   false
  		 DeleteSets     false
  		 TimerPercentile 90.0
	</Plugin>


The data from collectd should now appear at the graphite web interface
